# MallService

在线网上商城（前台、后台） 

## 项目目录结构

```
├─src
    ├─main
    │  ├─java
    │  │  └─site
    │  │      └─linkway
    │  │          ├─core
    │  │          │  ├─controller（控制器）
    │  │          │  ├─entity  （实体类）
    │  │          │  │  ├─bo  
    │  │          │  │  ├─po  （mybatis）  
    │  │          │  │  └─vo  （视图层实体类(JSON)）
    │  │          │  ├─filter  （拦截器）
    │  │          │  ├─mapper  （mapper类）
    │  │          │  └─service （服务类）
    │  │          └─utils  （常用工具类）
    │  ├─resources
    │  │  ├─mapper（mapper 相关xml文件）
    │  │  ├─mybean  （自定义bean）  
    │  │  └─sql   (数据库初始化SQL) 
    │  └─webapp  （前端项目）
    │      └─WEB-INF
```

## 相关技术
Spring、SpringMVC、Mybatis 、MySQL
